# Basic-Calculator-function-built-in-python
This project is a basic command-line calculator built using Python. It demonstrates the use of user-defined functions, conditional statements, and user input handling to perform common arithmetic operations.

# Features

## Addition
## Subtraction
## Multiplication
## Division
## Average of two numbers

#  Concepts Used

## Python functions
## Conditional statements (if-elif-else)
## User input (input())
## Basic arithmetic operators
## Clean and modular code structure
